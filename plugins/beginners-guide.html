Beginner's Guide

Start here for After Effects setup, plugins, composition settings, and shortcuts.

1. Getting Started with After Effects

Adobe After Effects is the industry standard for motion graphics and visual effects. Before installing plugins, make sure your AE version is up to date. Most modern plugins require AE 2023 or later.

  • Download After Effects from Adobe Creative Cloud
  • Keep your old AE version installed until all projects, effects, and presets are confirmed working in the new version
  • Set your composition settings correctly: 1920x1080, 30fps or 60fps, square pixels

2. Installing Plugins

Plugins extend After Effects with new effects and capabilities. Here's how to install them:

  1. Download the plugin from Zyrex Plugins
  2. Extract the ZIP file using 7-Zip (Windows) or The Unarchiver (Mac)
  3. Copy the plugin files (.aex for Windows, .plugin for Mac) to the AE Plug-ins folder:
    C:\Program Files\Adobe\Adobe After Effects [version]\Support Files\Plug-ins\ (Windows)
    /Applications/Adobe After Effects [version]/Plug-ins/ (Mac)
  4. Restart After Effects
  5. Find your new plugin under the Effects & Presets panel
Tip: Create a dedicated folder for third-party plugins to keep things organized. This makes it easier to troubleshoot if issues arise.

3. Essential Keyboard Shortcuts

  • Space — Play/Pause preview
  • Ctrl+D / Cmd+D — Duplicate layer
  • P — Position property
  • S — Scale property
  • T — Opacity property
  • U — Show all keyframed properties
  • Ctrl+Shift+C / Cmd+Shift+C — Pre-compose
  • Ctrl+M / Cmd+Ctrl+M — Add to Render Queue

4. Common Issues & Fixes

Plugin not showing up? Make sure you placed it in the correct folder and restarted AE. Check that the plugin version matches your AE version.

Project files won't open? If a project depends on older plugins, check Plugins for compatible versions or visit the Discord for help.

Performance issues? Enable Disk Cache in Preferences → Media & Disk Cache. Set it to at least 50GB on a fast SSD.
