Guides

Written setup, troubleshooting, activation, plugin, and workflow guides.

Setup Guides

  • Changing Your After Effects Version — Keep your old AE version installed until projects, effects, and presets are confirmed working. Newer plugin installers often expect AE 2024+.
  • Optimizing AE Performance — Enable Multi-Frame Rendering, set Disk Cache to 50GB+ on SSD, allocate sufficient RAM in Preferences → Memory.
  • Setting Up a Clean AE Workspace — Recommended panel layout: Project (left), Composition (center), Effects & Presets + Effect Controls (right), Timeline (bottom).

Troubleshooting Guides

  • Plugin Not Loading — Verify the plugin is in the correct folder. Check AE version compatibility. Try running AE as administrator.
  • Project Crashes on Open — Hold Shift while opening AE to bypass loading plugins. Remove recently added plugins one by one to isolate the issue.
  • Missing DLL Errors (Windows) — Install Visual C++ Redistributable. Some plugins require specific VC++ versions.
  • Gatekeeper Blocks Plugin (Mac) — Go to System Preferences → Security & Privacy → Allow the blocked plugin. Or use xattr -cr in Terminal.

Activation Guides

  • Always extract with 7-Zip — The built-in Windows extractor may cause errors. Download 7-Zip.
  • Read the included README — Most downloads include a readme.txt or .nfo file with specific activation instructions.
  • Run as Administrator — Some activators and patchers require admin rights. Right-click → Run as Administrator.
  • Temporarily disable antivirus — False positives are common with repacked software. Add an exclusion for your plugin folder.

Workflow Guides

  • Organizing Your Plugin Library — Create category folders: Effects, Transitions, Color, Text, Utilities. This speeds up your workflow.
  • Preset Management — Save custom presets via Effects & Presets panel → Save Animation Preset. Store them in a dedicated folder.
  • Expression Basics — Learn essential AE expressions: wiggle(freq, amp), loopOut(), time*value. They can replace many plugin effects.
