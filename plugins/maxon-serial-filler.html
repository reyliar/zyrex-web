Maxon Serial Filler Method

License Red Giant, Universe, and Trapcode plugins using Maxon App 2024.5.0 and the Serial Filler utility.

When to Use This

This is an alternate licensing path when the standard patch or service unlocker isn't working. It relies on Maxon App 2024.5.0 (specifically — not newer), plugin builds from before 2023.0, and Maxon's built-in Serial Filler tool. For missing effects and service errors, check the Maxon Fixes guide first.

Do not accept Maxon App updates. Newer versions break this method. If you accidentally update, clean and reinstall 2024.5.0.

Windows Setup

  1. Clean uninstall. Remove everything Maxon-related. Use the Maxon cleaner script if a normal uninstall leaves leftovers.
  2. Install Maxon App 2024.5.0. Get the specific Windows installer — decline any update prompts.
  3. Sign in. Create a free Maxon account if you don't have one.
  4. Install plugins. For each product, choose Other versions and select a build before 2023.0. Compatible generations: Magic Bullet 15–16, Trapcode 7–8, Universe 5–6, VFX Suite 2–3.
  5. Run Serial Filler. Located in C:\Program Files\Maxon\Tools\. Enter the serial for each plugin — one at a time.

macOS Setup

  1. Exit RLM mode. In Maxon App, choose Exit RLM Mode (top right) and sign in.
  2. Uninstall Red Giant plugins. Remove Magic Bullet, Trapcode, Universe, etc. from Maxon App. Do not uninstall Cinema 4D — AE depends on it.
  3. Uninstall Maxon App. Remove it if the cleaner didn't already.
  4. Install Maxon App 2024.5.0 for macOS. Decline updates, sign in.
  5. Install plugins via Other versions. Same pre-2023.0 rule as Windows.
  6. Run Serial Filler. From Maxon App's tools, license each installed product.

Serial Numbers

Enter these in Serial Filler for the matching product:

  • Magic Bullet Suite: COBK2245921573563861 (try COBK2245846563534111 if first fails)
  • Universe: RLPK2245855360592319
  • Trapcode Suite: TCBK2245868172939255
  • VFX Suite: VXXF1545921787095659
  • PluralEyes: PFEK1245921362842515
